**PIA de Criptografía Equipo 1**
Este repositorio contiene:
  - Manual de usuario: Un manual que indica al usuario el modo de uso de nuestro sistema de firmas digitales
  - Documento explicatorio: Documento para la evaluación de este proyecto
  - signature.exe: Un ejecutable empaquetado portable de la versión de consola de este nuestro sistema, hecho para su ejecución en caso de no contar con Python o sus librerías instalados.
  - signature.py: La versión de consola de nuestro sistema escrita en Python, para la revisión de su lógica y desarrollo.
  - signature_gui.exe: Un ejecutable empaquetado portable de la versión gráfica de nuestro sistema.
  - signature_gui.py: La versión gráfica de nuestro sistema escrita en Python, para la revisión de su lógica y desarrollo.
